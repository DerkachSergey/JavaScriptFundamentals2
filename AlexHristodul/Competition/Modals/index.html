Подробное руководство по подключению:
Bootstrap "modal"


Давайте рассмотрим различные части загрузки модального окна, чтобы понять лучше.
Всплывающее окно Bootstrap "Modal" делится на три основных раздела:

  1. Заголовок,
  2. Тело,
  3. Нижний колонтитул.

Каждый раздел имеет свое собственное значение и, следовательно, должны использоваться надлежащим образом. Преимущество всплывающего окна Bootstrap "Modal", в том что Вам не придется писать JavaScript для применения его для своего сайта. Весь код и стили в Bootstrap "Modal" стандартные. Все что нужно для работы, это правильно использовать вызовы, разметку и атрибуты, это не трудно, читайте дальше и Вы узнаете как это сделать.

Подключение

Чтобы вызвать модальное окно на веб-странице сайта, вам необходимо вывести на страницу ссылку или кнопку. Разметка для вызова модального окна:

<!-- Кнопка вызова модального окна--> <a class="btn btn-lg btn-success" href="#" data-toggle="modal" data-target="#basicModal" id="modalShow">Открыть модальное окно </a>

Обратите внимание на ссылку, элемент <a> имеет два собственных атрибута данных: data-toggle и data-target. Переключатель data-toggle указывает Bootstrap, что ему нужно открыть всплывающее окно, а data-target указывает на элемент для открытия. Поэтому, после нажатия на кнопку появится модальное окно с идентификатором “basicModal”. Теперь давайте посмотрим на код, необходимый для определения модального окона:

<!-- Разметка модального окна --> <div class="modal fade" id="basicModal" tabindex="-1" role="dialog"> <div class="modal-dialog"> <div class="modal-content"> <div class="modal-header"> <!-- Кнопка закрытия --> <button class="close" type="button" data-dismiss="modal">x</button> <!-- Название модального окна --> <h4 class="modal-title" id="myModalLabel">Название модального окна</h4> </div> <!-- Содержимое модального окна --> <div class="modal-body"> <h3>Содержимое модального окна</h3> </div> <div class="modal-footer"> <!-- Кнопка закрытия --> <button class="btn btn-default" type="button" data-dismiss="modal">Закрыть </button> <!-- Кнопка которая возвращает изменения --> <button class="btn btn-primary" type="button">Сохранить изменения</button> </div> </div> </div> </div>

В корневом элементе <div> модального окна должен быть одинаковый идентификатор, который используется в качестве атрибута data-target.В нашем примере это идентификатор id="basicModal". Пользовательские атрибуты, такие как aria-labelledby и aria-hidden в корневом элементе используются для доступности. В коде HTML, существует блок, вложенный в корневой элемент <div>. Он имеет класс modal-content и говорит bootstrap.js где искать содержимое для модального окна. Внутри этого элемента <div>, мы должны разместить три блока,которые я упоминал ранее: заголовок, тело и нижний колонтитул. Заголовок модального окна, как следует из самого названия, используется для названия всплывающего окна и некоторые другие элементы, такие как “x” — закрыть. Кнопка закрыть имеет атрибут data-dismiss, который указывает какой элемент закрыть. После заголовка Вы можете добавлять любые данные внутри дочернего элемента <div>, в том числе видео с YouTube, вставлять изображения, или что-нибудь другое.В конце модального окна нижний колонтитул. По умолчанию эта область с левой стороны. В этой области можно было бы поместить действие кнопки “Сохранить”, “Закрыть”,“Принять”, и др.,которые связаны с действием отображения модального окна.

Изменение размеров модального окна

Bootstrap Modal адаптивный для всех размеров мониторов. В этом разделе Вы увидите, как изменить размер всплывающего окна. Комплект Bootstrap Modal поставляется с двумя новыми вариантами окна: Большое и Маленькое. Для большего окна нужно добавить модификатор класса modal-lg к modal-dialog <div>, а для маленького окна modal-sm.

Настройка с помощью jQuery

Modal — это плагин jQuery, поэтому, если вы хотите модальное окно с использованием jQuery, Вам необходимо вызвать функцию .modal() на модальные селектора. Например:

<!-- Вызов модального окна прямо в файле HTML --> <script type="text/javascript"> $(document).ready(function(){ $(".btn").click(function(){ $("#myModalonJS").modal('show'); }); }); </script> <!-- Вызов модального окна в отдельном файле .js --> $(document).ready(function(){ $(".modalShow").click(function(){ $("#myModalonJS").modal({ // конфигурация modal backdrop : true, // затемнение веб-страницы; keyboard : false, // использование ESC; remote : false, // показ содержимого удалёного URL; show : true // отображает модальное окно; }); }); });

Параметры метода "modal()"

Имя Тип Описание
backdrop boolean или string 'static' Значение по умолчанию: true.
Если установить значение параметра backdrop равное true, то происходит наложение темного фона над всем содержимым веб-страницы, поверх которого отображается модальное окно. Данный эффект используется по умолчанию, для его отмены установите значение backdrop равное false. У данного параметра backdrop есть дополнительное значение 'static', которое запрещает закрывать модальное окно при нажатии за его пределами. Данный параметр также можно установить с помощью data атрибута data-backdrop, например: data-backdrop="static", data-keyboard="false".
keyboard boolean Значение по умолчанию: true.
Закрывает модальное окно при нажатии на клавиатуре клавиши Esc.
Данный параметр также можно установить с помощью data атрибута data-keyboard.
load Url или boolean Используется для удаленной загрузки содержимого с помощью jQuery метод load(). Вам необходимо указать внешнею страницу в этот параметр. Значение по умолчанию false.
show boolean Значение по умолчанию: true.
Отображает модальное окно.
Данный параметр также можно установить с помощью data атрибута data-show

События

Модальные окна Bootstrap включают в себя несколько событий для расширения их функциональности.

Имя Описание
show.bs.modal Событие срабатывает, когда происходит вызов метода show() для отображения модального окна. Пример: $('#idmodal').on('show.bs.modal', function () { // сделать что-нибудь… })
shown.bs.modal Событие срабатывает, когда модальное окно становится видимым для пользователя, т.е. оно полностью нарисовано с помощью стилей CSS. Пример: $('#idmodal').on('shown.bs.modal', function () { // сделать что-нибудь… })
hide.bs.modal Событие срабатывает, когда происходит вызов метода hide(), который предназначен для скрытия модального окна. Пример: $('#idmodal').on('hide.bs.modal', function () { // сделать что-нибудь… })
hidden.bs.modal Событие срабатывает всякий раз, когда модальное окно становится полностью невидимым для пользователя, т.е. завершилось применение всех CSS свойств выполняющих этот процесс. Пример: $('#identifier').on('hidden.bs.modal', function () { // сделать что-нибудь… })

Удаленное содержимое в модальном окне

Существует три различных способа подключить удаленное содержимое внутри всплывающего окна Bootstrap "Modal". Первый способ заключается в использовании remote опция внутри объекта(как представлено выше), как упоминалось выше. Два других способа сделать это без JavaScript, как показано ниже. Вы можете указать ссылку href на внешнее содержимое всплывающего окна:

<a class="btn btn-lg btn-default" data-toggle="modal" data-target="#remoteModal" href="remote-page.html">Открыть модальное окно </a>

или использовать пользовательский атрибут данных data-remote вместо href.
Например:

<a class="btn btn-lg btn-default" data-toggle="modal" data-target="#remoteModal" data-remote="remote-page.html">Открыть модальное окно </a>
