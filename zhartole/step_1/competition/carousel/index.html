Описание Что такое Слайдер

Слайдер - это в основном автоматическое отображение изображений (слайд-шоу), он также может использоваться для отображения анонса последних новостей, завершенных проектов, описаний услуг, ссылок на последние статьи из блога.

Слайдеры очень популярные компоненты, большенство разработчиков используют их в своей работе.

Сейчас мы рассмотрим, как создать слайдер при помощи компонента Carousel в Bootstrap 3 и 4.

Преимущества Слайдера Bootstrap Carousel

Нет необходимости писать JavaScript код для использования слайдера на своем сайте

Код а так же стили в Bootstrap стандартные

Для работы с Bootstrap Carousel нужно только правильно использовать вызовы, разметку и атрибуты.

Придерживаясь нашей инструкции Вы убедитесь что Carousel от Bootstrap это легко настраиваемый и адаптивный слайдер.

  2. Управляем каруселью при помощи data атрибутов

    Для начала давайте разберемся какие атрибуты за что отвечают.

    data-slide имеет значения prev и next, которые соответственно изменяют позицию слайда относительно его текущего позложения. data-slide обычно используеться для кнопки, при нажатии на которую, происходит переход на предыдущий или следующий слайд.
    data-slide-to атрибут который используеться для перехода к любому слайду карусели при помощи соответствующего индекса. Например: data-slide-to="1" - переход к слайду с индексом 1.
    data-ride="carousel" этот атрибут нужен для того чтоб после загрузки страницы происходила автоматическая смена слайдов.

    Так же при помощи класа carousel-caption внутри любых .item мы имеем возможность с легкостью добавлять подпись на наших слайдах.

    Параметры Слайдера Карусели

    Параметры можна передать через атрибуты или JavaScript. Если передавать через атрибуты, тогда необходимо добавить приставку с названием параметра data, например: data-interval="3000".

    Название Тип Значение по умолчанию Описание
    interval number 5000 Время в миллисекундах между автоматической сменой слайдов карусели.
    pause string "hover" Делает паузу при нахождении курсора в пределах карусели. Возобновляет показ когда курсор за пределами слайда.
    wrap boolean true Запрещает непрерывный переход на следующий слайд с последнего слайда (т.е. он должен быть первым) и на предыдущий слайд с первого слайда (т.е. он должен быть последним)
  3. Управляем каруселью при помощи JavaScript

    Есть возможность вызвать карусель вручную с помощью JavaScript кода: $('.carousel').carousel()
    Этот код состоит из метода .carousel() и id либо класса к которому собственно и нужно применить данный метод.

    Метод .carousel() который предназначен для управления слайдером имеет свои стандартные методы.

    Метод Описание Пример
    .carousel(options) Инициализирует карусель с дополнительными параметрами и запускает процесс смены слайдов $('#id').carousel({interval: 4000})
    .carousel('cycle') Запускает процесс смены слайдов слева направо $('#id').carousel('cycle')
    .carousel('pause') Останавливает процесс автоматической смены слайдов карусели $('#id')..carousel('pause')
    .carousel(number) Осуществляет переход на определённый слайд карусели, который устанавливается с помощью числа number (отчёт ведётся с 0, по аналогии с массивом) $('#id').carousel(1)
    .carousel('prev') Осуществляет переход на предыдущий слайд $('#id').carousel('prev')
    .carousel('next') Осуществляет переход на следующий слайд $('#id').carousel('next')

    События с каруселью

    Чтобы изменить состояния карусели можно воспользоваться такими событиями:

    Событие Описание Пример
    slide.bs.carouse Событие срабатывает перед началом процесса смены слайда 
    $('#id').on('slide.bs.carousel', function () {
    // сделать что-нибудь... })
    slid.bs.carousel Событие срабатывает после процесса смены слайда 
    $('#id').on('slid.bs.carousel', function () {
    // сделать что-нибудь...})
