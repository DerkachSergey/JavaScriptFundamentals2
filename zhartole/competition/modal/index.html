Описание Что такое Всплывающее или Модальное (pop-up) окно

Всплывающее окно (pop-up окно) — окно, которое открываеться на экране в результате исполнения какой-либо операции (действия). Сегодня модальные окна широко используються на множестве сайтов

Модальные окна на веб-странице применяются для решения множества задач.

Например, всплывающее окно может использоваться для отображения пользователю формы входа и регистрации, изображения, сообщения о каком-либо событии, и прочьей информации.

Также плюсом модальных окон являеться то, что браузеры не могут их блокировать, как всплывающие окна, поэтому они отлично подойдут для концентрации внимания пользователей.

Простые в использовании и легко настраевымые всплывающие (модальные или pop-up) окна для Вашего проекта обеспечит Компонент Modal фреймворка Bootstrap 3 и 4.

Bootstrap Modal используеться для отображения на веб-странице контента с помощью всплывающего окна. В Bootstrap 3 и 4 всплывающее (модальное) окно представляет собой блок, который визуально отображается над прочим содержимым страницы.

Bootstrap Modal это один из самых полезных плагинов представленных фреймворком Bootstrap 3 и 4.

  2. Активируем модальное окно

    Сначала активируем модальное окно без единой строчки JavaScript кода. Для этого Устанавливаем аттрибуты data-toggle="modal", data-target="#myModal" или href="#myModal" на какой-либо html элемент, предположим на кнопку.

    data-toggle="modal" - атрибут, который содержит значение modal которое в свою очередь говорит, что содержимое, которое вызвает этот элемент, - модальное окно.
    #myModal - селектор (идентификатор) елемента который будет "всплывать".

    Ниже приведён код работающего примера. Для просмотра работающего примера нажмите на кнопку в правом верхнем углу.

    Параметры Модального окна

    Параметры можна передать через атрибуты или JavaScript. Если передавать через атрибуты, тогда необходимо добавить приставку с названием параметра data, например: data-backdrop="".

    Название Тип Значение по умолчанию Описание
    backdrop логический или строковый 'static' true При установке значение элемента backdrop равным true, темный фон налаживаеться над всем содержимым веб-страницы, поверх которого и отображается наше модальное окно. Этот эффект включен по умолчанию. Если хотите его снять, тогда установите значение backdrop равное false. У параметра backdrop есть ещё значение 'static', которое не дозволяет закрывать модальное окно при нажатии за его пределами.
    keyboard логический true Закрывает всплывающее окно при клике по клавише Esc.
    show логический true Отображает всплывающее окно сразу же после его инициализации.
    remote путь false Если указан url, то jQuery подгрузит его методом load и вставит ответ в элемент с классом .modal-body. При использовании data api, Вы сможете указать url в теге href, например: 
    <a data-toggle="modal"href="remote.html" data-target="#modal">Click me</a>
  3. Открываем модальное окно с помощью JavaScript

    Для того чтоб открыть модальное окно посредством JavaScript вызываем метод .modal() и укажем идентификатор (#id) либо класс (.class) элемента-контейнера, который содержит HTML-код модального окна. 

    $(document).ready(function(){
//при нажатии на любую кнопку, имеющую класс .btn
$(".btn").click(function() {
//открыть модальное окно с id="myModal"
$("#myModal").modal('show');
});
});

    JavaScript Метод modal() используеться не только для активации но и для управления модальным окном

    Осмотрим детальней метод .modal() который предназначен как для активации содержимого модального окна, так и для его управления.

    Имя Описание
    .modal(параметры) Активирует контент, указанный через #id или .class как модальное окно. В качестве параметров принимает значения: backdrop, keyboard, show. Например: 
    $('#myModal').modal({
backdrop: 'static',
keyboard: false
});
    .modal('toggle') Вручную переключает модальное окно. Это значит если всплывающее окно открыто, то скрывает его. А если он скрыто, то отображает его. Например: 
    $('#myModal').modal('toggle');
    .modal('show') Вручную открывает всплывающее окно. Например: 
    $('#myMmodal').modal('show');
    .modal('hide') Вручную скрывает всплывающее окно. Например: 
    $('#myMmodal').modal('hide');

    Для примера, активируем всплывающее окно для элемента c id #launch-modal:

    HTML CSS JavaScript
    <!-- HTML код кнопки --> <input id="launch-modal" type="button" class="btn btn-primary" value="открыть модальное окно"> <!-- HTML-код модального окна--> <div id="myModal" class="modal fade" style="text-align:justify;"> <div class="modal-dialog"> <div class="modal-content"> <!-- Заголовок модального окна --> <div class="modal-header"> <button type="button" class="close" data-dismiss="modal" aria-hidden="true">×</button> <h4 class="modal-title">JS Helper - твой путеводитель в мире JavaScript</h4> </div> <!-- Содержимое модального окна --> <div class="modal-body"> <p>Мы в JS Helper собрали для Вас лучшие решения неоднократно испытаны на практике.</p> <p>Многолетний опыт работы наших специалистов поможет Вам разобраться в концепциях JavaScript.</p> <p>С нами Вы обязательно сделаете свой первый шаг на пути к профессии JavaScript developer.</p> </div> <!-- Футер модального окна --> <div class="modal-footer"> <button type="button" class="btn btn-default" data-dismiss="modal">Упустить шанс</button> <button type="button" class="btn btn-primary">Стать JavaScript разработчиком</button> </div> </div> </div> </div> 
    
$(document).ready(function(){
//подпишемся на событие click элемента c id="#launch-modal"
	$('#launch-modal').click(function() {
//активируем контент c id="myModal", как модальное окно
		$('#myModal').modal({
//установим всплывающему окну следующие параметры:
			backdrop: 'static', /* Забыли что это значит?! Смотрим више!)  */
			keyboard: true
		});
	});
});

    События с модальным окном

    Чтобы изменить состояния всплывающего окна можно воспользоваться такими событиями:

    Имя Описание
    show.bs.modal Событие происходит, когда вызывается метод show(), предназначенный для открытия модального окна. Если модальное окно вызывается с помощью клика (click), то элемент, который вызвал открытие модального окна, будет доступен как свойство relatedTarget события.
    Например: 
    $('#myModal').on('show.bs.modal', function() {
			// Что-то сделать...
			})
    shown.bs.modal Событие происходит, когда модальное окно становится видимым для пользователя, т.е. когда оно будет полностью отрисовано с помощью стилей CSS. Если модальное окно вызывается с помощью клика (click), то элемент, который вызвал открытие модального окна будет доступен как свойство relatedTarget события.
    Например: 
    $('#myModal').on('shown.bs.modal', function() {
			// Что-то сделать...
			})
    hide.bs.modal Событие происходит, когда вызывается метод hide(), который предназначен для скрытия модального окна.
    Например: 
    $('#myModal').on('hide.bs.modal', function() {
			// Что-то сделать...
			})
    hidden.bs.modal Событие происходит, когда модальное окно становится полностью невидимым для пользователя, т.е. когда будут отработаны полностью все CSS-свойства, выполняющие этот процесс.
    Например: 
    $('#myModal').on('hidden.bs.modal', function() {
			// Что-то сделать...
			})
  4. Темизируем модальное окно

    Как вы уже знаете создание и управление видом модального окна в Bootstrap 3 и 4 осуществляется через классы CSS, атрибуты data- и методы JavaScript.

    Для того чтоб стилизировать модальное окно, необходимо разобраться в самой структуре всплывающего окна.

    Модальное окно состоит из таких блоков:

    • каркас - 3 блока div, имеющие классы .modal, .modal-dialog и .modal-content.
    • header (заголовок) - блок div, имеющий класс .modal-header.
    • body (основное содержимое) - блок div с классом .modal-body.
    • footer (футер) - блок div с классом .modal-footer.

    Ниже размещен HTML код всплывающего окна. Вам осталось только включить воображение и воплотить все свои идеи в жизнь с помощью знакомых CSS & HTML. Например что-то такое:

    HTML код модального окна, в самом простом его представлении, выглядит следующим образом:

    <div id="myModal" class="modal fade"> <div class="modal-dialog"> <div class="modal-content"> <!-- Заголовок --> <div class="modal-header"> <button type="button" class="close" data-dismiss="modal" aria-hidden="true">×</button> <h4 class="modal-title">Заголовок модального окна</h4> </div> <!-- Содержимое --> <div class="modal-body"> Содержимое модального окна... </div> <!-- Футер --> <div class="modal-footer"> <button type="button" class="btn btn-default" data-dismiss="modal">Закрыть</button> <button type="button" class="btn btn-primary">Сохранить изменения</button> </div> </div> </div> </div>

Скрипт, который открывает всплывающее окно каждые 5 минут после загрузки страницы:

HTML CSS JavaScript
<!-- Модальное окно --> <div class="modal fade" id="myModal" tabindex="-1" role="dialog" aria-labelledby="myModalLabel" aria-hidden="true"> <div class="modal-dialog" role="document"> <div class="modal-content"> <div class="modal-header"> <button type="button" class="close" data-dismiss="modal" aria-label="Закрыть"> <span aria-hidden="true">×</span> </button> <h4 class="modal-title" id="myModalLabel">Заголовок модального окна</h4> </div> <div class="modal-body"> Содержимое модального окна... </div> <div class="modal-footer"> <button type="button" class="btn btn-default" data-dismiss="modal">Закрыть</button> <button type="button" class="btn btn-primary">Сохранить изменения</button> </div> </div> </div> </div> 

$(window).load(function() {
// вызываем метод setInterval, который будет вызывать модальное окно каждые 5 минут, если оно не открыто
setInterval(function() {
// Если окно не открыто (т.е. не имеет класс in)
if (!$("#myModal").hasClass("in")) {
// то открыть модальное окно
$("#myModal").modal('show');
}
}, 300000);
});
