"Игральные кости"

Создать HTML документ с елементом:
<div id="result"></div>

Подклчить к нему внешний скрипт.

Написать цикл for, который пройдет 15 итераций.

Внутри цикла на каждой итерации цикла необходимо генерировать два случайных числа, 
для этого воспользуйтесь следующим кодом:
var first = Math.floor((Math.random() * 6) + 1); 
var second = Math.floor((Math.random() * 6) + 1); 

В каждой итерации цикла необходимо выводить содержимое этих переменных
в формате: "Первая кость:" + first + "Вторая кость:" + second;

Выводить необходимо в див с помощью команды:
document.getElementById("result").innerHTML += "Первая кость:" + first + "Вторая кость:" + second + "<br>";

Дополнительно напечатать такую информацию:
Если переменные обе одинаковы дополнительно вывести "Выпал дубль. Число <number>"
document.getElementById("result").innerHTML += "Выпал дубль. Число <number>"

Используя условные операторы(|| &&) если оба числа равны 1 то вывести 
"Две единицы"

Если оюа числа равны 6 то вывести 
"Две шестерки"

В каждой итерации вычислять общую сумму первой и второй переменной и прибавлять к общему количеству.

Снаружи цикла (после его завершения) с помощью тернарного оператора проверить переменную total.
Если она больше 100 вывести "Победа, вы набрали <score> очков" иначе "Вы проиграми, у вас <score> очков";

Схема:

var total = 0
Цикл {
     вычисляем переменные каждый рас заново, 
     печатаем на экран
     печатаем совпадения если есть
     печатаем если есть две единицы или две шестерки

     total += first + second;
}
Проверяем total с помощью оператора(? :) и выводим результат.
