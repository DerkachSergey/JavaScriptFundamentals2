Задание

Необходимо сделать 2 задания.

Первое.

Есть input и textarea. Необходимо на input установить обработчик событий на keyup. При наборе текста в левом поле (input) автоматически на лету текст переноситься в textarea посимвольно.
В случае стирания кода из input текст textarea тоже должен стираться посимвольно
При нажатии на кнопку "Стереть" оба поля очишаются.




Второе.
Необходимо взять галерею из прошлого задания (версия которая с createEmement) и добавить следующий функционал.


Простая версия.

Убираете фильтрацию по количеству и по диапазону.
Добавляете кнопочку "Добавить новый элемент". При нажатии на эту кнопочку берется следующая картинка из массива и отрисовывается.
В шаблоне надо добавить кнопочку "Удалить", при нажатии на кнопочку соответствующая картинка удаляется.

Продвинутая версия.

Кроме всего этого добавляются три input поля в которых надо указать диапазон от, до, и количество внутри диапазона.
Инпуты можно взять из предыдущего примера. Логика вырезания такая же как и в предыдущем примере.
